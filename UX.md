# UX.md — myapt

## Product feel
복잡한 금융 계산을 숨기지 않되, 사용자에게는 짧고 명확하게 설명하는 도구.

## Main flow
1. 홈에서 재무 정보 + 지역 입력
2. 즉시 예산 계산 결과 확인
3. 같은 흐름에서 검색 결과 확인
4. 결과 필터 조정
5. 단지 카드에서 상세 요약 확인 후 네이버 이동

## Screens
### Home / Search
- 상단 서비스 설명
- 재무 입력 폼
- 대출 시나리오 선택
- 기본 지역 입력
- 계산 실행 버튼

### Result
- 요약 카드 4개
  - 최대 예산
  - 추정 대출 가능액
  - 월 상환액
  - 적용 규칙 요약
- 적용 제한 사유 카드
- 필터 바
- 결과 리스트 또는 테이블

### Admin
- 규칙 버전
- 금리/취득비용 설정
- 최근 동기화 상태
- 수동 매핑 관리 표

## UX rules
- 금융 용어에는 짧은 설명을 같이 붙인다.
- “왜 이 한도인지”가 중요하다.
- 결과는 모바일에서도 카드형으로 읽히게 한다.
- 실거래가 stale / 거래 적음 상태를 눈에 띄게 표시한다.
